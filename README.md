## Live demo

> This demo app is deployed at <a href="https://crm-frontend-nuxt.qrinef.ru" target="_blank">crm-frontend-nuxt.qrinef.ru</a>

## Installation

```bash
$ npm install
```

## Running the app

```bash
# serve with hot reload at localhost:3000
$ npm run dev

# build for production and launch server
$ npm run build
$ npm run start
```
